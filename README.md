# dumpAnywhere64
An IDA (Interactive Disassembler) script that can save a chunk of binary from an address.

## Usage:
 * Example: dumpAnywhere64("/Users/GeoSn0w/Desktop/file.bin",   0xFFFFFF,     64)
                                          |                        |          |
                                File path where to save     Start Address    Size
